## ⚙ Instalando

### Clonando o repositório
```bash
git clone https://github.com/Cavalheiro-S/split-money-server-nest
```

### Entrando na pasta do repositório
```bash
cd split-money-server-nest
```

### Instalando pacotes
```bash
npm i
```

### Iniciando o projeto
```bash
npm run dev
```
